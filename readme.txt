== Description ==

This plugin retrives the last post you saved/published and 
directly from the Post edit page allows you to share it with others on your Google Buzz

== Installation ==

1. Upload Folder 'GoogleBuzzAdmin' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are Ready :)
